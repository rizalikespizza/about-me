What I've made

HLNoobs.net (Half Life Noobs) April 2023 - Ongoing

HLNoobs.net is a website, game servers, and a discord server. The goal of this project is to preserve, document, and host GoldSrc and Source engine content. A lot of info for creating content and hosting servers for games on these game engines are scattered to the wind and back so I figured something like this would be useful for everyone to have easy access to. The website is basic but it gets the job done. From there if I recieve a donations to host dedicated servers for specific games we have the info for or know where decent info is hosted, I make that server. This is my first major project. It has taught me the basics of web development, how to use Linux CLI, security practices, and how to run + maintain servers.
