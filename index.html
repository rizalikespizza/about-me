What I've made

HLNoobs.net (Half Life Noobs) May 2023 - Ongoing

HLNoobs.net is a website, game servers, and a discord server. The goal of this project is to preserve, document, and host GoldSrc and Source engine content. A lot of info for creating content and hosting servers for games on these game engines are scattered to the wind and back so I figured something like this would be useful for everyone to have easy access to. The website is basic but it gets the job done. From there if I recieve a donation to host dedicated servers for specific games we have the info for or know where decent info is hosted, I make that server. This is my first major project. It has taught me the basics of web development, how to use Linux CLI, security practices, and how to run + maintain servers.

What software I recommend

Linux Mint GNU/Linux Distribution

Linux Mint is what I use instead of Windows to get things done. I've messed around with a ton of different distros at this point and have come to the conclusion I want things to just work for the most part when I want to get work done. It's distracting and takes up time when things break on you and Linux Mint does an amazing job at being up to date enough while staying stable.

VSCodium Visual Studio Code without Microsoft

VSCodium is my current IDE (Integrated Development Environment) of choice. It's what I use to code. If you dislike Microsoft but like VSCode then you will like this too.

Kdenlive FOSS Video Editor

Kdenlive is what I use to edit videos. I used to use Premier Pro and Resolve but I prefer the free and open source nature of Kdenlive.

GIMP GNU Image Manipulation Program

GNU Image Manipulation Program is a FOSS alternative to programs like Photoshop. It's not as advanced as Photoshop but is more then capable for basic image manipulation needs.
